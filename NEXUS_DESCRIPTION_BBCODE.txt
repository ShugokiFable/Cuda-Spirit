[heading]Description[/heading]
Cuda Spirit is a [b]self-contained Windows app[/b] for new adventurers, returning veterans, and long-time players who are tired of guessing.

Black Desert scatters rewards across a dozen inboxes, hides binding and transfer rules, traps people into irreversible item choices and character deletion mistakes, and fills the Pearl Shop with offers that look better than they are. Cuda Spirit is built for that mess: [b]deterministic safety engines[/b], a versioned local SQLite knowledge store, current public-source sync, returner recovery, and optional [b]retrieval-focused AI guidance[/b]. Unknown or stale facts fail closed instead of becoming confident guesses.

Version [b]2.4.1 · Obsidian UI[/b] replaces the older pill-heavy look with a sharper matte OLED cockpit: crisp geometry, restrained accents, five live themes, compact navigation, global [b]Ctrl+K[/b] command palette, and a reworked HUD / advisor overlay.

[b]One self-contained exe. No .NET install for end users. Not affiliated with Pearl Abyss.[/b]

[heading]Privacy & anti-cheat safety[/heading]
Cuda Spirit is a [b]companion and decision-support app[/b]. It is designed to stay external to the game client.
[list]
[*][b]ZERO memory reading[/b] of [code]BlackDesert64.exe[/code] or any game process.[/*]
[*][b]ZERO DLL injection, hooks, or client modification.[/*]
[*][b]ZERO packet sniffing or anti-cheat evasion.[/*]
[*][b]ZERO input automation / botting / unattended gameplay.[/*]
[*]Data comes from [b]public pages and APIs[/b], [b]manual entry[/b], [b]local JSON imports you place[/b], and [b]optional DXGI screen capture[/b] you trigger.[/*]
[*]Network traffic only happens for features you enable (public BDO pages/APIs, optional BDO Alerts, optional profile/Garmoth pages, OpenRouter when AI is used).[/*]
[*]OpenRouter prompts may include your question, retrieved knowledge, saved profile fields, open Navigator tasks, and recent decision summaries. [b]Do not paste passwords, payment info, support tickets, or recovery codes.[/b][/*]
[*]Local data lives under [code]%APPDATA%\CudaSpirit[/code] ([code]settings.json[/code], [code]cudaspirit.db[/code], [code]imports\[/code]). API keys are stored in local settings and should be treated as secrets. Public releases never ship your database, keys, imports, or screenshots.[/*]
[/list]


[heading]Installation & setup[/heading]
[list=1]
[*]Download and extract [code]Cuda-Spirit-2.4.1-Nexus-win-x64.zip[/code].[/*]
[*]Run [code]CudaSpirit.exe[/code] (or [code]RUN CUDA SPIRIT.bat[/code]) from any folder [b]outside Program Files[/b].[/*]
[*]No .NET runtime or SDK is required for this self-contained release. WebView2 Runtime is used for official-page fallback and the embedded browser (preinstalled on Windows 11; Microsoft Evergreen Runtime on most Windows 10 systems).[/*]
[*]Open [b]Customize[/b]: set region, stage, focus, weekly hours, spending style, budget, and owned conveniences.[/*]
[*](Optional) On the Dashboard / recovery flow, paste your [b]official BDO adventurer profile link[/b] and sync. Optionally fetch public Garmoth enrichment when available.[/*]
[*](Optional) Open [b]Settings[/b] and paste an [b]OpenRouter[/b] API key only if you want AI Advisor / Vision Scan.[/*]
[*]Open [b]Returner & Reroll Recovery[/b] if you are coming back; use [b]Rewards[/b], [b]Item Intel[/b], [b]Transfer Lab[/b], and [b]Pearl Shop Guard[/b] before irreversible actions.[/*]
[*]Toggle the [b]Overlay[/b] when you want a floating HUD. Prefer Borderless / Windowed game display for window tracking.[/*]
[/list]
[i]Core recovery, safety, calculators, trackers, and data workspaces run without an AI key. AI is optional context on top of local evidence — not a requirement to use the app.[/i]

Delete the extract folder / [code]CudaSpirit.exe[/code]. Remove [code]%APPDATA%\CudaSpirit[/code] only if you also want to erase settings, the local knowledge DB, imports, and stored keys.

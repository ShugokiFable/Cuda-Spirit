# Privacy and account safety

Cuda Spirit stores its database and settings locally under `%APPDATA%\CudaSpirit`.

## Data that can leave the computer

Only features the user enables make network requests:

- public Black Desert pages and APIs;
- optional BDO Alerts API using the user’s key;
- optional public adventurer/Garmoth profile pages;
- OpenRouter prompts when AI features are used.

OpenRouter prompts may include the current question, relevant retrieved knowledge, saved gameplay profile, open Navigator tasks, and recent decision summaries. Do not paste passwords, payment information, private support tickets, or account recovery codes.

## Data the app does not access

Cuda Spirit does not read game process memory, capture packets, send input, inspect hidden account data, bypass anti-cheat, modify the client, or automate gameplay.

## Backups and reset

Back up `%APPDATA%\CudaSpirit` to preserve settings, tasks, history, and the knowledge database. Delete that folder to reset local application data. API keys are stored in the local settings file and should be treated as secrets.
